# 專案需求與策略：曜亞 x 默默的社群經營

> **2026 年度核心目標**：建立 **Neuramis (仙女)**、**Cooltech Define (酷特)**、**LPG Infinity (法式)** 三大品牌的 B2B 社群領導地位。

---

## 1. 經營目標 (Project Goals)
1.  **出圈 (Outreach)**：拓展觸及到陌生客群，擴大品牌影響力。
2.  **黏著度 (Retention)**：穩定增長既有受眾 (TA)，維持高互動率。
3.  **異業合作 (Collaboration)**：保持內容彈性，配合公司進行跨產業或品牌合作。

---

## 2. 每月產出與頻率 (Deliverables & Frequency)
依據「醫美品牌 Facebook 粉絲專頁代操需求規格表」：

| 項目 | 頻率 | 內容說明 |
| :--- | :--- | :--- |
| **圖文貼文 (Posts)** | **18 篇 / 月** | 每個品牌各 6 篇。每週產出約 4-5 篇。 |
| **限時動態 (Stories)** | **30 則 / 月** | 每個品牌各 10 則。維持每日一篇動態的活躍度。 |
| **短影音 (Reels)** | **6 支 / 月** | 每個品牌各 2 支。6-15秒 (AI/動畫) 或 15-30秒 (資訊)。 |
| **內容企劃 (Planning)** | **每月 1 式** | 制定當月主題與排程。 |
| **數據報表 (Report)** | **每月 1 份** | 成效分析與優化建議。 |

---

## 3. 內容策略配比 (Content Strategy)
**總計：每月 18 篇貼文 (平均分配給 Neuramis、Cooltech、LPG)**
*   **各品牌配額**：
    *   **貼文**：6 篇
    *   **限動**：10 則
    *   **短影音**：2 支
*   **知識性 (Knowledge)**：30-40%。
    *   *重點*：材料介紹、醫學知識、「痛點/解決方案」。
    *   *目標*：建立信任感與權威性 (科學專家)。
*   **生活/話題 (Lifestyle)**：60-70%。
    *   *重點*：節慶話題、時事跟風、氛圍感、品牌價值。
    *   *目標*：提升互動、親切感、品牌形象 (頂級體驗/閨蜜)。

### 3.1 品牌溝通與視覺風格 (Brand Communication & Visuals)

#### 🔴 **Neuramis 仙女玻尿酸 (The Fairy)**
*   **溝通核心**：強調「自然」、「純淨」、「SHAPE技術」。
*   **視覺調性**：**柔和粉 / 純淨白** (Soft, Pure, Fairy)。
*   **關鍵字**：仙女光、安心變美、韓國銷量霸主。

#### 🔵 **Cooltech Define 酷特冷凍減脂 (The Sculptor)**
*   **溝通核心**：強調「效率」、「360度」、「4探頭」。
*   **視覺調性**：**科技藍 / 冰晶白** (Ice, Tech, Sharp)。
*   **關鍵字**：Define Your Best、忙碌首選、不只是減脂。

#### 🟡 **LPG Infinity 法式體雕 (The Muse)**
*   **溝通核心**：強調「第11代無限波」、「細胞級運動」、「身心平衡」。
*   **視覺調性**：**香檳金 / 簡約白** (Elegant, Flow, Infinity)。
*   **關鍵字**：法式優雅、被動運動、身心靈對話。

---

## 4. 短影音風格 (Short Video Styles)
1.  **AI / 動畫型**：8-20 秒。使用 **Google Veo 3.1** 生成高視覺衝擊、抽象或科技感畫面 (如：Cooltech 冰晶流動)。
2.  **資訊 / 產品型**：15-30 秒。教育性內容、產品展示、訊息清晰。

## 5. 法規與審查 (Compliance & Legal)

> ⚠️ **重要**：完整合規指南請參閱 [CONTENT_COMPLIANCE_GUIDE.md](./CONTENT_COMPLIANCE_GUIDE.md)

### 🚫 絕對禁止的用語

| 類型 | 禁用詞彙 |
|------|---------|
| **療效宣稱** | 刺激脂肪代謝、促進膠原蛋白、改善淋巴循環、緊緻肌膚、消水腫、撫紋、減脂、燃脂 |
| **2C 行銷** | 讓妳變美、馬上諮詢、限時優惠、Before/After 對比 |
| **療程用語** | 療程、治療、患者、術後 |
| **銷售導向** | 銷量第一、必買、最有效、保證效果 |

### ✅ 替代用語建議

| 原本 | 替代 |
|------|------|
| 療程 | 課程、服務、體驗 |
| 減脂 | 體態管理、曲線規劃 |
| 術後 | 使用後、居家照護 |
| 效果 | 技術特點、設計亮點 |

*   **流程**：提交初稿 -> 合規檢查 -> 品牌審核 -> 排程上架 (需預留 3 個工作天緩衝)。
